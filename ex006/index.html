Ordered lists

Use "ol" and "li" for ordered lists

  1. First item
  2. Second item
  3. Third item
  4. Fourth item

You can also start with a different number

  1. Second item
  2. Third item
  3. Fourth item

You can also use type="a":

  1. First item
  2. Second item
  3. Third item
  4. Fourth item

Or type="i":

  1. First item
  2. Second item
  3. Third item
  4. Fourth item
